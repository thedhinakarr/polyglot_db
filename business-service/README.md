
# Business Service

This service handles core business logic for the Polyglot Persistence System.

## Features

- Core business operations
- Transaction management
- Business rules validation

## Technologies

- Rust with Actix-Web
- PostgreSQL for relational data
- MongoDB for document storage

## Setup

1. Build the project:
   ```bash
   cargo build
   ```

2. Create `.env` file (use `.env.example` as a template when available)

3. Run the service:
   ```bash
   cargo run
   ```

## Testing

```bash
cargo test
```
